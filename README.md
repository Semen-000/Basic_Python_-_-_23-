# 🍵 Задача 1.Решение задачи "Чаепитие у Алона"
# 📌 Условие
Алон живёт на последнем этаже

Лифт вмещает 4 человека

Нужно посчитать только ПОЛНЫЕ поездки вверх (ровно 4 человека)

Переменная friend_count уже дана (число друзей от 1 до 100)

# 💡 Мои рассуждения
Понял, что нужно найти, сколько раз в лифт зайдут ровно 4 человека

Это просто целое количество четвёрок в общем числе друзей

Вспомнил про целочисленное деление //

# ✅ Решение
```bash
trips_up = friend_count // 4

print(trips_up)
```
# 📊 Примеры
7 друзей → 7 // 4 = 1 поездка (4 + 3)

4 друга → 4 // 4 = 1 поездка (4)

3 друга → 3 // 4 = 0 поездок (все в одной, но неполной)

# 🎯 Итог
Задача оказалась простой — достаточно было внимательно прочитать условие и вспомнить про целочисленное деление.

# ⏱️ Задача 2.Решение задачи "Тайны времени"
# 📌 Условие
Дано число секунд (переменная number)

Нужно перевести в формат "минуты и секунды"

Вывести два числа через пробел: минуты и секунды

# 💡 Мои рассуждения
В одной минуте 60 секунд

Чтобы найти минуты — нужно разделить общее число секунд на 60 и взять целую часть

Остаток от деления на 60 — это оставшиеся секунды

Вспомнил про операции // (целочисленное деление) и % (остаток)

# ✅ Решение
```bash
minutes = number // 60

seconds = number % 60

print(minutes, seconds)
```
# 📊 Примеры
120 секунд → 120 // 60 = 2 минуты, 120 % 60 = 0 секунд → вывод: 2 0

65 секунд → 65 // 60 = 1 минута, 65 % 60 = 5 секунд → вывод: 1 5

30 секунд → 30 // 60 = 0 минут, 30 % 60 = 30 секунд → вывод: 0 30

# 🎯 Итог
Классическая задача на перевод секунд в минуты. Главное — не перепутать, что выводится сначала минуты, потом секунды.

# 📌 Условие

```alon``` — первая цифра

```van_dama``` — вторая цифра

```stal_one``` — третья цифра

```jecki_kachan``` — четвёртая цифра

```chack_no_rise``` — пятая цифра

# 💡 Мои рассуждения
Число можно превратить в строку, тогда каждая цифра станет отдельным символом

Потом каждый символ превратить обратно в число

Так мы точно не перепутаем порядок

# ✅ Решение
```bash
cake_str = str(cake_number)

alon = int(cake_str[0])

van_dama = int(cake_str[1])

stal_one = int(cake_str[2])

jecki_kachan = int(cake_str[3])

chack_no_rise = int(cake_str[4])

bruslya = int(cake_str[5])
```

cake_str = "123456"

alon = 1          # cake_str[0] = "1"

van_dama = 2      # cake_str[1] = "2"  

stal_one = 3      # cake_str[2] = "3"

jecki_kachan = 4  # cake_str[3] = "4"

chack_no_rise = 5 # cake_str[4] = "5"

stal_one = 4      # cake_str[2] = "4"

bruslya = 1       # cake_str[5] = "1"

# 🎯 Итог
Самый простой и понятный способ — через строку. Не нужно думать про остатки от деления и порядок получения цифр с конца.

# 🧮 Решение задачи "Скобки Алона"
# 📌 Условие
У Алона есть код с математическими выражениями, но без скобок они считаются неправильно. Нужно расставить скобки, чтобы получить нужные результаты.

# 💡 Мои рассуждения
## 🔹 Выражение a = 20 + 20 * 2 → нужно 80
### Как работает без скобок:

- Сначала умножение: 20 * 2 = 40

- Потом сложение: 20 + 40 = 60 ❌

### Как получить 80:

- Нужно сначала сложить, потом умножить

- Ставим скобки на сложение: ```(20 + 20) * 2```

- 20 + 20 = 40, 40 * 2 = 80 ✅

## 🔹 Выражение b = 2 ** 2 + 1 → нужно 8
### Как работает без скобок:

- Сначала возведение в степень: 2 ** 2 = 4

- Потом сложение: 4 + 1 = 5 ❌

### Как получить 8:

- Нужно сначала сложить, потом возвести в степень

- Ставим скобки на сложение: ```2 ** (2 + 1)```

- 2 + 1 = 3, 2 ** 3 = 8 ✅

## 🔹 Выражение c = 3 * 3 ** 2 + 1 → нужно 81
### Как работает без скобок:

- Сначала возведение в степень: 3 ** 2 = 9

- Потом умножение: 3 * 9 = 27

- Потом сложение: 27 + 1 = 28 ❌

### Как получить 81:

- Нужно перемножить тройки и возвести в квадрат: ```(3 * 3) ** 2```

- 3 * 3 = 9, 9 ** 2 = 81 ✅

## 🔹 Выражение d = -2 ** 2 * -2 → нужно -8
### Как работает без скобок:

- Унарный минус имеет высокий приоритет

- -2 ** 2 = -(2**2) = -4

- Умножение: -4 * -2 = 8 ❌

### Как получить -8:

- Нужно сначала возвести в квадрат само число -2

- Ставим скобки на -2: ```(-2) ** 2```

- (-2) ** 2 = 4

- Потом умножаем на -2: 4 * -2 = -8 ✅

# ✅ Итоговое решение
```bash
a = (20 + 20) * 2     # (40) * 2 = 80

b = 2 ** (2 + 1)      # 2 ** 3 = 8

c = (3 * 3) ** 2      # 9 ** 2 = 81

d = (-2) ** 2 * -2    # 4 * -2 = -8

print(a, b, c, d)
```
# 📊 Проверка результатов
| Выражение | Было | Стало |
|-----------|------|-------|
| a | 60 | **80** |
| b | 5 | **8** |
| c | 28 | **81** |
| d | 8 | **-8** |
# 🎯 Итог
Скобки меняют порядок действий! Вспомнил приоритет операций:

Скобки ```()```

Возведение в степень ```**```

Унарный минус ```-```

Умножение/деление ```* / // %```

Сложение/вычитание ```+ -```
