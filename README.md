# 🍵 Решение задачи "Чаепитие у Алона"
# 📌 Условие
Алон живёт на последнем этаже

Лифт вмещает 4 человека

Нужно посчитать только ПОЛНЫЕ поездки вверх (ровно 4 человека)

Переменная friend_count уже дана (число друзей от 1 до 100)

# 💡 Мои рассуждения
Понял, что нужно найти, сколько раз в лифт зайдут ровно 4 человека

Это просто целое количество четвёрок в общем числе друзей

Вспомнил про целочисленное деление //

# ✅ Решение
python
```bash
trips_up = friend_count // 4

print(trips_up)
```
# 📊 Примеры
7 друзей → 7 // 4 = 1 поездка (4 + 3)

4 друга → 4 // 4 = 1 поездка (4)

3 друга → 3 // 4 = 0 поездок (все в одной, но неполной)
