# Функция print(): Шпаргалка
## Аргументы
Объекты, которые передаём в функцию (строки, числа, списки)

Разделяются запятой

Ошибка при отсутствии запятой

# Параметры (имеют значения по умолчанию)
## Параметр end
Значение по умолчанию: "\n" (перевод строки)

Добавляется после последнего аргумента

Пример: print("Привет", end=" ") print("Мир") → "Привет Мир"

## Параметр sep
Значение по умолчанию: " " (пробел)

Разделяет аргументы между собой

Пример: print("Я", "ты", "он", sep="-") → "Я-ты-он"

# Совместное использование sep и end
print("A", "B", "C", sep="...", end="!!!") → "A...B...C!!!"

# Использование переменных
razdelitel = " | "
print("Раз", "Два", "Три", sep=razdelitel) → "Раз | Два | Три"

# Важные моменты
sep и end указываются в конце вызова функции

Запись без пробелов: sep='_' (соответствует PEP 8)

Принимают только строковые значения

Не путать /n (слеш) и \n (обратный слеш)

# Сводка
| Параметр | Область действия | Значение по умолчанию |
|----------|-----------------|----------------------|
| sep | Между аргументами | пробел " " |
| end | После последнего аргумента | перевод строки "\n" |
